title: How do I connect a new ad network?
description: How to integrate a third-party ad network into ATSystems.
---

## Overview

ATSystems supports direct integration with external ad networks via their APIs.

## Steps

1. Go to **Integrations** in your ATSystems dashboard
2. Click **Add Network**
3. Select the network from the list (or choose Custom)
4. Enter the required API credentials (key, secret, etc.)
5. Save and test the connection

## Notes

- Each network may need different credentials
- Success message means data syncing has started
